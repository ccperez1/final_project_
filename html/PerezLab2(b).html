Terms Definitions and Schedule Table

A table is used to arrange data in a form of rows and columns. It is used for tabular data so it will be easier to find the fixed information/data.

HTML Elements

tr element

  • It stands for table row. It is used when you want to create a new row in a table. It is the second tag you would use after table element so you can determine how many rows you would want.

th element

  • It stands for Table Heading and it is used when you would want to to create a heading cell. It is one of two type of cells that a table will have. The th element is found after the use of table and tr element.

td element

  • The td element stands for table data. It is the second type of cell that you can create on a table beside table header cell. It is used under the table heading cell because it contains data and information based on what it is writing on the heading.

tfoot element

  • It stands for a table frooter. It is used afetr the table data so it you can write separate information that you want to add on the table. It creates an ending section row for the table.

caption element

  • It is a table caption that is used to create a small title for the table.

Difference between colspan and rowspan

  1. Colspan- Merges columns together(horizontally). It also spans the colums allowing them to be more space within as many colums together.
  2. Rowspan- Merges rows together(vertically). It also spans the rows to become more spacious to fit more information in the rows.
  3. They both merge and combine text boxes but the aligned line from the data is different.

Difference between cells Spacing and padding

  1. Padding is a property used to generate space around an element content. It is used to create space inside a cell so it is not in a tight space. You are also able to use properties to specify what side you want to add space too. You could also add the styling to state the amount of space you want.
    2. Some examples would be:
    • padding-top: 40px;
    • padding-right: 30px;
    • padding-bottom: 20px;
    • padding-left: 50px;
  2. Cell Spacing is a property that creates space between the heading, and data cells.
    • border-spacing: 20px;

Define table border using CSS

It is property that is able to control the thickness of th border you will have in you table. You are also able to style your border with dashes.

How do you control border widths using CSS?

It is a property that able to control the width of the table border. You would type border-width: thin and it will change the size of the border to thin.
